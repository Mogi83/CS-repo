  • In 2 cases above, which case tends to wander farther, and why?

I think the best way to describe each case is to look at the expected value and variance of each walk.

As a personal reminder, here are the formulas for expected value and variance; note that because we have the entire population, we use this formula for variance.

$$E[X] = x_1p_1 + x_2p_2 + ... + x_kp_k$$

$$Var(X) = E[X^2] - (E[X])^2$$

Expected Values¶

Taking a look at expected values for walk1 there is an equal probability of getting either a 1 or -1 so $p_1$ and $p_2$ are both $0.5$, $x_1 = $ and $x_2 = -1$

$$E[X] = (1 * 0.5) + ((-1) * 0.5)$$ $$E[X] = 0.5 + (-0.5) = 0$$

This tells us that we wouldn't expect our random walk to trend very far from $0$.

Looking the expected value of walk2 our values for x are $[-2,-1,1,3]$ each with an equal probability of being selected of, $0.25$, therefore:

$$E[X] = ((-2) * 0.25) + ((-1) * 0.25) + (1 * 0.25) + (3 * 0.25)$$ $$E[X] = (-0.5) + (-0.25) + 0.25 + 0.75$$ $$E[X] = (-0.75) + 1 = 0.25$$

With an expected value of $0.25$ I would expect a positive trend to be apparent in walk2

Variance¶

walk1: To find the variance of walk1 we have to find $E[X^2]$ first:

$$E[X^2] = (0.5 * (1)^2) + (0.5 * (-1)^2)$$ $$E[X^2] = 0.5 + 0.5 = 1$$

Therefore:

$$ Var(X) = 1 - 0^2 = 1 $$

walk2: $$E[X^2] = (0.25 * (-2)^2) + (0.25 * (-1)^2) + (0.25 * 1^2) + (0.25 * 3^2) $$ $$E[X^2] = (0.25 * 4) + (0.25 * 1) + (0.25 * 1) + (0.25 * 9) $$ $$E[X^2] = 1 + 0.25 + 0.25 + 2.25 = 3.75 $$

Therefore:

$$ Var(X) = 3.75 - (0.25)^2 $$ $$ Var(X) = 3.75 - 0.0625 = 3.6875 $$

All of this to demonstrate that the variance of walk2 is higher than walk1 so we would expect more spread from walk2.
